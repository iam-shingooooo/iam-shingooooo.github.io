2025年の学習の振り返り

  • 木 25 12月 2025
  • Certifications and Learning

はじめに

2025年は離職期間にあったため、勤務先で即時活用することを前提とした学習ではなく、自身のキャリア目標に即した学習に注力した。
具体的には、ITゼネラリストとして社会的に認知されるスキルセットを習得するため、幅広い分野を横断的に学習する一年となった。
なお、英語については生涯を通じて継続的に学ぶべきものと考えており、当記事では触れていない。

資格

今年は以下の資格を取得した。

  • Microsoft Certified: Dynamics 365 Fundamentals (ERP)
  • Linux入門（LFS101-JP）
  • メンタルヘルス・マネジメント検定試験Ⅲ種

Microsoft Certified: Dynamics 365 Fundamentals (ERP)を取得したことで、Microsoft認定資格のFundamentalsレベルの資格をすべて取得するという目標を達成でき、一定の達成感を得られた。
また資格というよりは認定バッジに近い位置付けではあるが、Microsoft Applied Skillsを11件取得した。

取得したMicrosoft Applied Skills

Microsoft Applied Skillsの取得目的は、AI関連のサービスや機能のキャッチアップ、Microsoft 365 Certified: Administrator Expert取得に向けた準備、Microsoft製品全体の復習である。
以下の資格については、現時点では取得には至っていないが、資格に対応する公式プログラムの受講などは完了しており、一定程度、資格取得に向けた準備が整っている。

  • Microsoft 365 Certified: Endpoint Administrator Associate
  • GitHub Foundations
  • ServiceNow Certified Implementation Specialist – Software Asset Management
  • ServiceNow Certified Implementation Specialist – Hardware Asset Management
  • CCNA

Microsoft 365 Certified: Administrator Expertを含め、これらすべての資格を取得しようとすると受験料が高額になる。
一方で、実務においては資格の保有そのものよりも、スキルを習得し、それを業務に活かして成果につなげることが重要である。
そのため、費用対効果や実務上の優先度などを考慮しつつ、取得を検討していく。

勉強会・ウェビナー

今年は以下のような分野の勉強会やウェビナーに参加した。

  • セキュリティ（AI、メール、サプライチェーン、SASE）
  • ネットワーク（SD-WAN、無線LAN、運用・保守）
  • AI（AIエージェント、MCP）
  • Microsoft製品（Dynamics 365、Power Platform、Purview、Active Directory、SharePoint、Windows 365）
  • 非IT分野（貿易実務、固定資産）
  • その他（IT戦略、AWS）

高い専門性が求められる分野については、個人での情報収集や学習には限界があるため、勉強会やウェビナーへの参加は有意義であったと感じている。
参加したイベント全体の傾向としては、セキュリティを主題としないイベントにおいても、セキュリティに関する話題が取り上げられるケースが比較的多かった。
また、AI については多くの参加者が利便性を認識している一方で、費用対効果やセキュリティ面の課題が多く、依然として発展途上の分野であるという印象を受けた。

おわりに

今年学習して得たスキルは、まだ業務に活かすことができていない。
来年は、今年習得したスキルを実務に活かし、具体的な成果をあげることを最優先の目標とする。
一方で、ITゼネラリストとして社会的に認知され、需要のある人材になれるように引き続き必要な学習を進めていく。
