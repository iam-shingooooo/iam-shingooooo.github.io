Microsoft Virtual Training Days 認定試験料50％OFF特典を受け取るための注意点

  • 水 06 8月 2025
  • Certifications and Learning

記事作成の背景

先日、Virtual Training Daysのイベントに参加したが、後述の注意点を把握しておらず、割引特典を受けられなかった。
本イベントの特典は基礎レベルの認定試験が対象で、本イベントの内容も初学者向けであるため、割引特典を受けたいと考えている人のなかには同社のイベントや認定試験に不慣れな人が一定数いると考えられる。
また、同社のイベントや認定試験に慣れている人であっても、私のように変更点を把握しておらず、割引特典を受けられないケースがあると想定し、そのようなケースを防ぐために注意点を共有する。

結論

Microsoft社が提供している本イベントに参加すると、イベントに対応した認定試験料が50％OFFになる特典が付与される。
その割引特典を受け取るためには、「Microsoftのアカウント」のメールアドレスでイベントに申し込む必要がある。

注意点

まず、以前は本イベント参加後に申込メールアドレス宛にバウチャーが届いていたが、現在はバウチャーレス方式に変更され、本イベントに申し込んだメールアドレスのアカウントで認定試験を申し込むと自動的に割引特典が適用される。
次に、本イベントの申し込みと参加は、Microsoftのアカウントのメールアドレスではなく、どのメールアドレスでも可能である。
しかし、割引特典の適用方式がバウチャーレス方式になったため、本イベント申込時のメールアドレスと認定試験申込時のアカウントのメールアドレスが一致していないと、割引特典が適用されない。
つまり、「Microsoftのアカウント」のメールアドレスで本イベントに申し込む必要がある。

イベントの申し込みページの抜粋:

既に Microsoft Learn アカウントをお持ちの場合、そのアカウントに関連付けているメール アドレスをイベント登録時に使用することをお勧めします。

Microsoft Virtual Training Daysの認定試験の割引に関するFAQの抜粋:

認定試験の割引を受けるにはどうすればよいですか? 手順に従ってイベント参加者認定試験割引を受けてください。サインイン時には、このイベントへの登録時に使用したメールアドレスを使用してください。Microsoft Learn アカウントを新規作成する場合は、イベントへの登録に使用したメール アドレスをMicrosoft Learn アカウントに関連付けてください。

「Microsoftのアカウント」についての補足

当記事における「Microsoftのアカウント」は、「職場または学校アカウント」と「Microsoftアカウント」の両方を指している。
認定資格は、Microsoft Learn プロファイルで管理され、このプロファイルはどちらのアカウントでも作成可能である。
